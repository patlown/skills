## HITL

These skills are human-in-the-loop workflows.

| Skill | Description |
| ----- | ----------- |
| [grill-me](./hitl/grill-me) | Interview the user relentlessly about a plan or design until reaching shared understanding. |
| [prototype](./hitl/prototype) | Build a throwaway prototype to answer a question before committing to a design. |
| [collab](./hitl/collab) | Land the first honest implementation of a real change in the existing codebase, then stop for review before broader cleanup. |
